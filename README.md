# Portfolio Template

A clean, responsive, and customizable personal portfolio template. Built with HTML, CSS, and vanilla JavaScript.

## Features

-   **Responsive Design**: Works on mobile and desktop.
-   **Dark/Light Mode**: User preference support with persistent storage.
-   **Clean Structure**: Organized assets for easy customization.
-   **SEO Friendly**: Basic meta tags included.

## Usage

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/portfolio-template.git
    cd portfolio-template
    ```

2.  **Customize:**
    -   Edit `index.html` to update your personal details (Name, Bio, Links).
    -   Replace `assets/images/profile.png` with your photo.
    -   Modify `assets/css/style.css` to change colors or fonts.

3.  **Deploy:**
    -   Upload to GitHub Pages, Netlify, or Vercel.

## Branch Protection

To ensure the integrity of the `master` branch, it is recommended to enable branch protection rules in the repository settings:

-   Require pull request reviews before merging.
-   Require status checks to pass before merging.
-   Include administrators.
-   Restrict who can push to matching branches.
